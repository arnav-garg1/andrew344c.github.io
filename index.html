Argus Security

A novel deep learning neural network placed in school security cameras to detect guns, alerting authorities and the community.

About Us

Our product utilizes real-time neural networks to create a responsive school security system that offers threat detection and flexibility in an emergency. A unique aspect of our product is our community alert system, which posts the location of the latest shooting on this interactive website to promote community safety.

Easy to Access

Completely public website allows the entire community to learn about the advanced weapon sighting, making sure that they avoid the location at all costs.

Fast and Reliable

Near-Instant recognition and alert system ensures that the teachers and authorities are informed of the situation as soon as possible. This allows for action to be taken as soon as possible, mitigating human error and saving countless lives.

Gun Detection Locations

Powered By

Tensorflow

We used TensorFlow to train our deep learning neural network, as it is a very reliable system, providing us great results.

Firebase

We utilize Google Firebase for two major parts of our project:

1. Firebase Database: We send the latitude and longitude of the location of the gun detection to our Firebase Database which then sends this data to our website which places this location on our interactive map.

2. Firebase Cloud Storage: When our deep learning neural network detects a gun, a picture of the sighting is sent to our Firebase Cloud Storage, which is only accessible privately by police offers, allows them to see if it truly is a gun detected and narrow down suspects.

Google Maps API

Utilizing the Google Maps API to provide viewers with an interactive map to allow them to easily see the location of every gun sighting.
